# Healthcare SQL Analysis Project

This project contains SQL scripts for creating and analyzing a healthcare billing database.  
The database simulates a medical billing environment with patients, providers, diagnosis codes, procedure codes, and encounters.

## Database Structure
The database includes the following tables:
- patients
- providers
- icd10_codes
- cpt_codes
- encounters (fact table)

## Analysis Performed
The SQL queries in this project perform various analyses, including:
- Patient demographics filtering
- Encounter and provider analysis
- Total paid amount by provider
- Monthly revenue trend
- Denied and pending encounter analysis
- High cost visit identification
- Running totals using window functions
- Provider ranking by revenue
- Latest encounters per provider
- Distinct patient counts per provider
- Top 5 highest paid encounters
- Pivot report for monthly revenue by provider
- Cumulative patient counts

## Skills Demonstrated
This project demonstrates the following SQL skills:
- Database creation and table design
- Primary and foreign keys
- Joins
- Aggregations
- Subqueries
- CASE statements
- Window functions
- Common Table Expressions (CTEs)
- Ranking and running totals
- Pivot-style reporting
